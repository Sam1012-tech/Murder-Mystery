Design a dark, cinematic, interactive UI for a murder mystery investigation website that feels like a real-time detective thriller movie.

Theme:
A secret police investigation system that the player (detective) has accessed. The interface should look like a classified digital case file database.

Style:
Dark mode UI, cinematic lighting, suspenseful atmosphere, glitch effects, subtle animations, hacker-style overlays, and investigative dashboards.

Color palette:
Black, dark grey, deep blue backgrounds with red highlights for danger and yellow highlights for clues.

Typography:
Use a mix of modern sans-serif and terminal-style fonts to create a hacker / investigation system aesthetic.

Create the following screens:

1. Intro / Access Screen
   A cinematic loading screen with glitch effects.
   Text appears like a system booting up.
   Example elements:
   "Initializing database…"
   "Decrypting files…"
   "Case File #731 Loaded"
   Button: "ACCESS CASE FILE"

2. Investigation Dashboard
   A futuristic police system dashboard.
   Panels for:

* Case file
* Evidence
* CCTV feeds
* Suspect database
* Terminal access
  Design it like a detective command center interface.

3. Crime Scene Investigation Screen
   Full screen crime scene image with clickable investigation markers.
   UI tools for:

* Zoom
* Evidence inspection
* Note-taking panel
* Clue highlights

4. CCTV Monitoring Screen
   A security camera monitoring interface with multiple camera feeds.
   Include:

* Camera grid layout
* Video playback controls
* Timestamp overlays
* "REC" indicator
* Static / CCTV effect

5. Incoming Phone Call Event Screen
   A dramatic phone call interface with a mysterious incoming call from an unknown number.
   Include:

* Caller ID: "UNKNOWN"
* Phone ringing animation
* Answer / Ignore buttons
* Call timer
* Dark cinematic style

6. Evidence Board Screen
   A detective-style evidence board with photos, documents, and clues connected by red threads.
   User can visually connect evidence pieces.

7. Suspect Interrogation Screen
   A chat-style interrogation interface where the detective questions suspects.
   Include message bubbles, suspect profile panel, and dialogue history.

Interaction ideas:

* Glitch transitions between screens
* Flickering lights
* Terminal typing animation
* Suspenseful UI animations
* Hover effects on clues
* Cinematic transitions

The overall design should feel immersive, mysterious, and interactive like a detective movie interface or investigative computer system.
